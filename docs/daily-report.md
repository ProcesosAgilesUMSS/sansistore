# Daily Report (On The Team Issue)

Each team has **one open GitHub issue** for the sprint. You do not create new daily issues.

Every day, **each member adds one comment** on that team issue.

## Format (copy/paste)

```markdown
- **Yesterday:** <what you did> (include commit SHAs / PR/Issue # if relevant)
- **Today:** <what you will do>
- **Blockers:** <None | describe + what you need>
```

## Example

```markdown
- **Yesterday:** worked on 4a65187, 61e5de3, 7fbc110, 1bc57ae
- **Today:** working on #7
- **Blockers:** None
```

## Referencing work

- Commits: short SHAs like `4a65187`.
- Issues and PR: `#8`.
